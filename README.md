# OCR-Java-Projet3
OpenClassrooms - Parcours DA Java/JEE - Projet 3

/*
** Concept de l'application
*/

Vous avez le choix entre 2 jeux : CodeSearch et Mastermind

Choix 1 : Le CodeSearch
Le but du jeu est de deviner ou de faire deviner une combinaison de 4 chiffres 

Choix 2 : Mastermind
Le but du jeu est de deviner ou de faire deviner une combinaison de 4 à 10 couleurs

Chaque jeu dispose de 3 modes :
- Challenger : vous devez deviner la combinaison
- Defender : l'ordinateur (NPC) doit deviner la combinaison
- Duel : A tour de rôle, le joueur et le npc doivent deviner la combinaison adverse
